# Stock Market Sentiment Analysis

This project predicts short-term stock market movement using financial news sentiment and machine learning models.

## Overview
Financial news headlines are analyzed to extract sentiment signals that influence stock price movement. Sentiment scores are combined with market data to classify next-day stock trends.

## Features
- Sentiment analysis of financial news
- Feature engineering for market movement prediction
- Machine learning models for classification
- Analysis of sentiment impact on stock trends

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- NLP techniques
- Jupyter Notebook

## Models
- Logistic Regression
- XGBoost
- LSTM (for time-series prediction)

## Goal
To explore how financial news sentiment can be used to improve short-term stock market prediction.
